# 🛠 vnint

**vnint** is a high-performance project scaffold generator designed for rapid development. It automates the creation of project structures, license integration via GitHub API, and configuration file setup.

[![img.png](https://i.postimg.cc/yN9qSPbf/img.png)](https://postimg.cc/4KJ2DzZt)          
            

## 🚀 Quick Start

### 1. Clone the repository

    git clone https://github.com/vn-ct/vnint.git cd vnint

### 2. Install dependencies

You will need requests and inquirer. Install them using pip: `pip install requests inquirer`

### 3. Run the generator

python vnint.py

## 📦 Features

-   Structure Scaffolding: Automatically creates /src, /tests, and /docs directories.
    
-   Live Licensing: Fetches up-to-date license templates directly from the GitHub API.
    
-   Configurable: Selectable extra files (Makefile, Docker, SECURITY.md, etc.).
    
-   Caching: Local caching for near-instant execution after the first run.

Built with precision for system-level automation and security-oriented workflows.
